# Research Explained — Agent Instructions

## Video Production

When producing multiple HyperFrames videos (3+):
1. Build ONE video first as a prototype
2. Present it for review
3. Iterate until approved
4. Use the approved version as the explicit quality reference in prompts for the remaining videos
5. Only then parallelize the rest

Do not mass-produce all videos in parallel on the first attempt.

## Visual Style

Follow `visual-style.md` in the project root for palette, font sizes, and content guidelines when creating any HyperFrames composition.
